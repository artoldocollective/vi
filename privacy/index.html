File di registro

Viaggio India segue una procedura standard per l'uso dei file di registro. Questi file registrano i visitatori quando visitano le pagine web. Tutti i provider di hosting fanno lo stesso e fa parte dei servizi di analisi di hosting. Le informazioni registrate dai file di log includono indirizzi IP (Internet Protocol), tipo di browser, provider di servizi internet (ISP), data e ora, pagine di riferimento/uscita e, eventualmente, il numero di clic. Questi dati non sono associati a informazioni che possano identificare personalmente gli utenti. Lo scopo di queste informazioni è l'analisi delle tendenze, la gestione del sito web, il monitoraggio dei movimenti degli utenti sul sito web e la raccolta di informazioni demografiche.

Cookie e Web Beacon

Come qualsiasi altro sito web, anche Viaggio India utilizza i "Cookie". Questi cookie sono utilizzati per memorizzare informazioni, tra cui le preferenze dei visitatori e le pagine web che il visitatore ha visitato. Le informazioni vengono utilizzate per ottimizzare l'esperienza dell'utente, adattando il contenuto del nostro sito web in base al tipo di browser del visitatore e/o altre informazioni.
